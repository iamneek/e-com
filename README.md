## Installation

Firstly, clone the repository.

```bash
$ git clone https://github.com/iamneek/e-com.git
```

Secondly, go inside the directory.
```bash
$ cd e-com
```

Finally, install dependencies using the following command

```bash
$ yarn
```

Now, run it using 

```bash
$ yarn dev
```
